## 🌑 THE MISSION
**VORTEX ALPHA** isn't just a tool; it's a high-performance system designed to map complex node relationships with zero latency. Built for the **Bajaj Finserv Engineering Challenge**, it delivers industrial-grade graph processing, cycle detection, and high-fidelity shader-enhanced visuals.

## ⚡ CORE CAPABILITIES
- **Deep Hierarchy Mapping**: Recursive tree construction from flat edge lists.
- **Cycle Intelligence**: Instant detection of infinite recursion and graph loops.
- **Apex Analytics**: Real-time summary of total trees, largest roots, and duplicate edges.
- **Premium Shader UI**: 100% aesthetic dark-mode interface powered by Framer Motion and Paper Shaders.
- **Zero-Failure Validation**: Strict node relationship enforcement (X->Y) with detailed error tracking.

## 🛠 THE TECH STACK
- **Engine**: Node.js / Express (The Backbone)
- **Pulse**: React / TypeScript (The Interface)
- **Brilliance**: Tailwind CSS v4 + Shadcn (The Aesthetics)
- **Motion**: Framer Motion + @paper-design/shaders-react (The Vibe)

## 🚀 ASCEND (SETUP)

### 1. CLONE THE REPO
```bash
git clone https://github.com/karanstark/VORTEX-ALPHA-.git
cd VORTEX-ALPHA-
```

### 2. POWER THE BACKEND
```bash
cd backend
npm install
node server.js
```
*Current Port: 5000*

### 3. IGNITE THE FRONTEND
```bash
cd ../frontend
npm install
npm run dev
```
*Current Port: 5200*
